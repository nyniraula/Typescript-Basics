# Setting up TypeScript

Install TS globally:
```bash
pnpm add -g typescript
```

Compile a TS file to JS:
```bash
tsc filename.ts
```
Or compile everything in the directory:
```bash
tsc
```

> [!NOTE]
> Even if TS shows compilation errors, the compiler still outputs the JS files by default.
> **Gotcha:** If you want to block compilation when errors exist, set `"noEmitOnError": true` in `tsconfig.json`.

### tsconfig.json
Basically configures what errors to show, warnings, and compiler rules for the project.
Generate it automatically:
```bash
tsc --init
```

### Auto-compilation on save
To avoid compiling manually every time:
```bash
tsc --watch
```
Watches for changes and auto-compiles on save.
