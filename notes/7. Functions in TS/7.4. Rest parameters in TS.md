# Rest parameters in TS

What is rest in JS?

Rest means just collect all the data passed as argument into an array.

In JS:

```javascript
function doSome(...rest) {
    console.log(rest) // [1,2,3,4,5,6,7,8,9]
}

doSome(1, 2, 3, 4, 5, 6, 7, 9)
```

In TS we have to define what kind of array `rest` will be.

```typescript
function doSome(...rest: number[]): void {
    console.log(rest) // [1,2,3,4,5,6,7,8,9]
}

doSome(1, 2, 3, 4, 5, 6, 7, 9)
```

or even for a mixed array:

```typescript
function doSomeMixed(...rest: (number | string)[]): void {
    // ...
}
```
