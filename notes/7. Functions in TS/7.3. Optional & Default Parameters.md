# Optional & Default Parameters

In TS, we define optional parameters with a question mark (`?`) after it.

```typescript
function doSomething(name: string, age: number, gender?: string): void {
    // ...
}
```

For default parameters we just give it a value like a variable declaration.

```typescript
function doSomethingElse(name: string, age: number, gender: string = 'undisclosed'): void {
    // ...
}
```

> **Gotcha**: Optional parameters (`?`) must always come *after* required parameters in the parameter list. Default parameters can be anywhere, but they are implicitly optional. Note that you don't need to use `?` if you're providing a default value, TS will automatically infer it's optional.
