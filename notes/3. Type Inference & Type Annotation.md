# Type Inference & Type Annotation

## Type Inference
Automatically figures out (infers) what type the variable is based on the data it is given. Always let simple variables be inferred by TS.

```typescript
let a = 23; // TS automatically infers that a is a number. No ': number' annotation required.
```

## Type Annotation
Annotating what type it is explicitly.

```typescript
let a: string = 'hello'; // ': string' is the type annotation.
```

### Function Annotation
```typescript
function name(param: type): return_type {
    // ...
}
```

That's all there is.

> [!WARNING]
> **Gotcha:** Once TS infers or you annotate a type, you cannot assign a value of a different type to even a non-constant variable. (For that, you have to explicitly use union types).
