# Generics with Classes

Generics can even work with classes.

You say, "I'll pass the property and its type when I instantiate the object, not fixed right now."

```typescript
class Bottle<T> {
    constructor(public name: string, public data: T) {}
}

const MiltonBottle = new Bottle<string>('Milton', 'Its durable');
const HiltonBottle = new Bottle<number>('Hilton', 32);
```

Or let it infer itself:

```typescript
const ChiltonBottle = new Bottle('Chilton', true);
```
