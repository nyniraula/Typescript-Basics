# Generics with Interfaces

We can even define object shapes with generics, and later provide the type variable.

Suppose you have a user object with `name`, `age` and `data`. You don't know what the type of `data` will be. You can just say "I'll pass the data type as I create that user," let's keep the type as a variable.

```typescript
interface name_of_interface<T> {
    key: T;
}

const newObj: name_of_interface<type> = { ... };
```

Example:

```typescript
interface User<T> {
    name: string;
    age: number;
    data: T;
}

const User1: User<string> = {
    name: "Adam",
    age: 25,
    data: "he is married"
};
```

Works! Now the data is a string that you passed as generics.

> **Note:** Inference doesn't work here with interface generics.
